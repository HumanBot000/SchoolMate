# SchoolMate Flutter Application Documentation

## Introduction

SchoolMate is a comprehensive mobile application designed to help students manage their academic
life efficiently. The app provides features for tracking schedules, homework, grades, and
school-related notifications. Built with Flutter and utilizing Supabase for backend services,
SchoolMate offers a modern, user-friendly interface for students to organize their academic
activities.

## Key Features

1. User Authentication
2. Schedule Management
3. Homework Tracking
4. Grade Tracking
5. Notification System
6. Personalized User Settings

## Development Environment Setup

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio or VS Code
- Android/iOS device or emulator

### Installation Steps

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Configure Supabase credentials in `API/supabase/setup.dart`
4. Run `flutter run` to launch the application
